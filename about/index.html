À propos

Dec 05, 2019
1 min read

Cloud Native Computing Paris est une association Loi 1901 visant à promouvoir les technologies Cloud Native. Le Cloud Native Computing permet de déployer des applications dans un environnement élastique et résilient.

Les technologies utilisées permettent l'orchestration de containers Linux dans un écosystème distribué. Ce groupe permet de partager nos expériences respectives autour de ces récentes solutions et concepts. Nous aborderons des sujets aussi divers que la virtualisation réseau, les meilleures pratiques de développement ainsi que toutes les solutions de “microservices”. Nous attendons des interventions des entreprises impliquées dans cette nouvelle pratique : Docker, Kubernetes, Hashicorp, Mesos, CoreOS, RedHat…

Plus d'information sur la Cloud Native Computing Foundation : https://cncf.io

Pour s'inscrire aux meetups et autres événements : https://www.meetup.com/fr-FR/Cloud-Native-Computing-Paris/
